The basic JavaScript Data Types
Type Description Examples Reference
Number Primitive (means it is not an object and has no methods). Represents integer or floating-point numbers between -(2^53 - 1) and 2^53 - 1 89 3.557 3e-2 29.42 JavaScript data types and data structures
BigInt The other JS primitive for numbers. This is for the big ones. It was coined recently to overcome the technical limitation of the Number type so it represents numbers bigger than 9007199254740991 and smaller than -9007199254740991. They created by appending a n to the end of an integer. const myBigInt = 9007199254740991000n;
String This primitive is the JavaScript's type for representing text. Actually an immutable ordered sequence of 16-bit values each of which represents a Unicode character. JavaScript: The Definitive Guide by David Flanagan
Boolean This primitive data type has two values, true or false. It is useful in controlling program flow using conditional statements like if..else, while..do something
"null" value Another primitive is the special value "null". It is usually used to indicate the absence of a value. It can be used to indicate "no value" for numbers, strings, and objects. It is a language keyword.
"undefined" value Also a primitive. Like the "null" value the "undefined" data type represents the absence of value but in a deeper way. If a variable have not been initialized, if an array or an object property not exist, a function has no return value JavaScript returns this value. It is a predefined global value. let year;
alert(year); // shows "undefined"
Objects and Symbols It's a fundamental datatype. An object is a composite value, aggregates multiple values (primitive values or other objects). It composed of pairs of keys and values. Objects include Arrays.
Symbol type is used to create unique identifiers for objects. The keys of the objects can be only string type or symbol type, nothing else.
typeOf operator This data type is an operator that returns a string indicating the type of the uvevaluated operand. typeOf true; // "boolean"
